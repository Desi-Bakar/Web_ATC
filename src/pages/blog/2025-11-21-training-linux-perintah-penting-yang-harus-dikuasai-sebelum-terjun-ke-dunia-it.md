#### Inti yang Perlu Dikuasai (Singkat & Praktis)

1. Menjelajah sistem

   * `ls`— lihat isi folder.
   * `cd`— pindah ke folder lain.
   * `pwd`— tahu di mana posisi kamu sekarang dalam struktur direktori.
2. Mengatur file dan folder

   * `mkdir`, `touch`— buat folder dan file baru.
   * `cp`, `mv`— salin atau pindahkan (atau ubah nama) file.
   * `rm`-`r`— hapus folder beserta isinya (hati-hati!).
3. Membaca dan mengubah file

   * `cat`, `less`— baca konten file.
   * `nano`(mudah) / `vim`(power user) — edit file konfigurasi.
4. Kontrol izin (permission)

   * `chmod`— atur siapa yang boleh baca/tulis/eksekusi.
   * `chown`— ganti pemilik file atau grupnya.
5. Memantau kondisi sistem

   * `top`/ `ps aux` — lihat proses yang berjalan.
   * `free -h` — cek penggunaan RAM.
   * `df -h` / `du`— pantau penggunaan disk.
6. Jaringan & konektivitas

   * `ping`— tes koneksi sederhana.
   * `ip a` — lihat alamat IP dan interface.
   * `ss -tuln` — cek port yang terbuka.
7. Layanan & otomatisasi

   * `systemctl`— jalankan/berhentikan/cek status service.
   * `crontab`— jadwalkan tugas berulang.

#### Kenapa Ini Penting untuk Karier IT?

Menguasai perintah-perintah di atas mempercepat troubleshooting, memudahkan deployment aplikasi, dan merupakan modal dasar bagi peran seperti System Administrator, DevOps, Cloud Engineer, atau Security Analyst.
