Saat mendengar kata *Cloud Computing* (Komputasi Awan), apa yang pertama kali terlintas di pikiranmu? Sebagian besar pemula mungkin akan langsung membayangkan Google Drive, iCloud, atau Dropbox.

Anggapan itu tidak salah, tetapi sebenarnya penyimpanan file (*cloud storage*) hanyalah sebagian kecil dari apa yang bisa dilakukan oleh teknologi Cloud Computing. Di era digital saat ini, Cloud Computing adalah mesin penggerak utama di balik aplikasi populer seperti Netflix, Spotify, hingga sistem *mobile banking* yang kamu gunakan setiap hari.

Melihat pesatnya adopsi teknologi ini di segala lini industri, *Areta Training Center* menghadirkan program kelas bertajuk **Bukan Sekadar Penyimpanan Online: Memahami Cara Kerja Cloud Computing bagi Pemula**. Kelas ini dirancang khusus dengan bahasa yang sederhana tanpa istilah teknis yang rumit, agar siswa, mahasiswa, dan pemula bisa paham luar-dalam tentang teknologi masa depan ini.

## Apa Sih Sebenarnya Cloud Computing Itu?

Bayangkan kamu ingin membuka usaha restoran. Daripada kamu membeli tanah, membangun gedung, dan membeli semua peralatan masak sendiri dari nol (yang membutuhkan modal sangat besar), kamu memilih untuk **menyewa** dapur siap pakai yang sudah lengkap dengan peralatannya. Kamu hanya perlu membayar biaya sewa bulanan sesuai dengan durasi pemakaian.

Itulah prinsip dasar Cloud Computing. Daripada perusahaan atau developer membeli server fisik yang mahal, menyimpannya di ruangan ber-AC khusus, dan merawatnya sendiri, mereka memilih untuk **menyewa daya komputasi** (server, database, jaringan, hingga software) milik perusahaan penyedia layanan cloud (seperti Amazon Web Services, Google Cloud, atau Microsoft Azure) melalui jaringan internet.

## 3 Model Layanan Cloud yang Wajib Diketahui Pemula

Dalam dunia Cloud Computing, layanan yang disediakan umumnya dibagi menjadi tiga kategori utama. Biar gampang diingat, mari gunakan analogi **"Pizza"**:

* **IaaS (Infrastructure as a Service):** Penyedia cloud hanya menyewakan infrastruktur dasar seperti server kosong dan jaringan digital. Kamu seperti *membeli bahan pizza mentah* (adonan, saus, keju), lalu kamu harus memanggang dan meraciknya sendiri di rumah.
* **PaaS (Platform as a Service):** Penyedia cloud menyiapkan server sekaligus lingkungan siap pakai untuk koding. Ini seperti layanan *delivery pizza mentah yang sudah diracik topingnya*, kamu tinggal memasukkannya ke dalam oven milikmu. Cocok untuk para developer aplikasi.
* **SaaS (Software as a Service):** Layanan aplikasi matang yang siap pakai langsung lewat browser tanpa perlu koding sama sekali. Ini ibarat kamu *makan pizza langsung di restorannya* atau memesan pizza yang sudah matang. Contoh nyatanya adalah Gmail, Canva, atau Netflix.

## Bagaimana Cara Kerja Cloud Computing?

Secara sederhana, cara kerja cloud computing dibagi menjadi dua bagian utama yang dihubungkan oleh internet:

1. **Front-End (Sisi Pengguna):** Ini adalah perangkat yang kamu gunakan (laptop, HP, atau tablet) beserta aplikasi atau browser yang digunakan untuk mengakses layanan cloud.
2. **Back-End (Sisi Penyedia):** Ini adalah pusat data (*data center*) raksasa yang berisi ratusan server fisik, komputer berspesifikasi dewa, dan sistem penyimpanan data yang berada di lokasi tersembunyi di berbagai belahan dunia.

Ketika kamu melakukan suatu perintah—misalnya mengedit video di Canva atau memutar film di Netflix—perangkatmu (Front-End) akan mengirimkan permintaan lewat internet. Server raksasa di *Back-End* akan memproses data tersebut menggunakan daya komputasi mereka yang super cepat, lalu mengirimkan hasilnya kembali ke layar HP-mu dalam hitungan milidetik. Hasilnya? HP-mu tidak cepat panas dan memorinya tidak penuh, karena semua proses berat dikerjakan di "Awan".

## Apa yang Akan Kamu Pelajari di Areta Training Center?

Di pelatihan ini, kamu tidak hanya belajar teori, tetapi juga simulasi langsung cara memanfaatkan efisiensi teknologi cloud:

* **Dasar-Dasar Komputasi Awan:** Memahami perbedaan mendalam antara server konvensional (On-Premise) dengan server berbasis Cloud.
* **Praktik Mengelola Cloud Storage & Share:** Trik berkolaborasi secara real-time menggunakan alat produktivitas berbasis cloud demi efisiensi kerja tim.
* **Pengenalan Layanan Cloud Populer:** Mengenal ekosistem Google Cloud Platform (GCP) dan AWS secara mendasar untuk pemula.
* **Keamanan Dasar di Cloud:** Cara mengamankan data dan akun cloud milikmu agar tidak mudah diintip atau diretas oleh orang lain.
