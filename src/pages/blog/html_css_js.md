## Kenapa Harus Belajar HTML, CSS, dan JavaScript?
HTML, CSS, dan JavaScript adalah tiga pilar utama standar web yang digunakan untuk membangun semua jenis website di internet. Dengan menguasai ketiganya, kamu akan bisa:

* Membuat struktur halaman web yang rapi dan ramah SEO.
* Mengatur tampilan visual website agar terlihat estetis dan responsif di HP maupun laptop.
* Memberikan interaktivitas dinamis agar website terasa hidup dan berfungsi dengan baik.
* Menjadi modal awal untuk berkarir sebagai Front-End Developer, Web Designer, atau membuka jasa pembuatan website.

## Apa yang Akan Kamu Pelajari?
Pelatihan di *Areta Training Center* mencakup materi dasar hingga praktik membuat produk web nyata, di antaranya:

* HTML (HyperText Markup Language): Mengenal tag dasar, menyusun teks, memasukkan gambar membuat tabel, serta menerapkan elemen semantik HTML5 yang baik untuk SEO.
* CSS (Cascading Style Sheets): Belajar mengatur warna, tipografi, efek visual, tata letak modern menggunakan Flexbox dan CSS Grid, hingga teknik desain responsivitas.
* JavaScript: Mengenal logika pemrograman dasar (variabel, fungsi, kondisi), manipulasi elemen web (DOM), serta memberikan fitur interaktif seperti tombol klik atau dark mode.

Semua materi disampaikan dengan metode *full praktik* dan didampingi oleh trainer berpengalaman di bidang pengembangan web.
